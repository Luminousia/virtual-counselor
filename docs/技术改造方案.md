## 一、项目概述

### 1.1 项目背景

本项目是一个基于 **React + Three.js + VRM** 的虚拟数字人心理咨询师游戏，数字人"小暖"扮演温和、耐心、热情的心理咨询师角色，与用户进行情感交流和心理支持。

### 1.2 现有功能

| 功能模块 | 描述 | 技术栈 |
|---------|------|--------|
| 3D数字人 | VRM格式3D角色模型渲染 | Three.js + @pixiv/three-vrm |
| AI对话 | 支持DeepSeek、OpenAI等多模型 | 流式API调用 |
| 语音合成 | 支持Genie-TTS、浏览器TTS、OpenAI TTS | 多TTS服务集成 |
| 口型同步 | 实时音频分析驱动口型 | Web Audio API |
| 表情系统 | 自动眨眼、表情变化、注视跟踪 | 动画控制器 |
| 状态管理 | 统一管理配置和状态 | Zustand |

### 1.3 改造目标

```
┌─────────────────────────────────────────────────────────────────┐
│                      改造目标                                    │
├─────────────────────────────────────────────────────────────────┤
│  🎯 平台迁移        Web端 → 桌面端（Electron）                    │
│  🎨 UI重构          现代化游戏化界面设计                          │
│  🎤 语音输入        集成语音识别，支持语音对话                     │
│  ⚡ 性能优化        启动速度、渲染性能、内存占用全面优化            │
```

### 2.3 现有UI问题

1. **布局单一**：左右分栏设计，缺乏游戏化沉浸感
2. **交互单调**：仅文本输入，缺少多媒体交互
3. **视觉扁平**：2D界面，与3D数字人融合度低
4. **反馈不足**：缺少操作反馈和状态提示
5. **移动端不友好**：固定布局，响应式差

### 2.4 现有对话流程

```
用户输入 → AI流式响应 → 文本分句 → TTS分段生成 → 音频队列播放
                                    ↓
                              口型同步驱动
```

#### 3.2.2 主进程实现 (`electron/main/main.ts`)

```typescript
import { app, BrowserWindow, ipcMain, Tray, Menu, globalShortcut } from 'electron';
import { createWindow, windowManager } from './windowManager';
import { registerIPCHandlers } from './ipc/handlers';
import { createTray } from '../plugins/tray';
import { registerShortcuts } from '../plugins/shortcut';
import { autoUpdaterService } from '../services/autoUpdater';

let tray: Tray | null = null;
let isQuitting = false;

app.whenReady().then(async () => {
  // 创建主窗口
  createWindow();
  
  // 注册IPC通信
  registerIPCHandlers();
  
  // 创建系统托盘
  tray = createTray();
  
  // 注册全局快捷键
  registerShortcuts();
  
  // 启动自动更新检查
  autoUpdaterService.checkForUpdates();
  
  // 平台特定优化
  if (process.platform === 'darwin') {
    app.dock.setIcon('./resources/icons/icon.icns');
  }
});

app.on('window-all-closed', () => {
  if (process.platform !== 'darwin') {
    app.quit();
  }
});

app.on('activate', () => {
  if (BrowserWindow.getAllWindows().length === 0) {
    createWindow();
  } else {
    windowManager.show();
  }
});

app.on('before-quit', () => {
  isQuitting = true;
});

// 单实例锁（防止多开）
const gotTheLock = app.requestSingleInstanceLock();
if (!gotTheLock) {
  app.quit();
} else {
  app.on('second-instance', () => {
    windowManager.show();
  });
}
```

#### 3.2.3 窗口管理 (`electron/main/windowManager.ts`)

```typescript
import { BrowserWindow, screen, Rectangle } from 'electron';
import * as path from 'path';

const isDev = process.env.NODE_ENV === 'development';

let mainWindow: BrowserWindow | null = null;
let isMaximized = false;
let lastWindowState: Rectangle = {
  x: 0,
  y: 0,
  width: 1400,
  height: 900
};

export function createWindow(): BrowserWindow {
  const { width, height } = screen.getPrimaryDisplay().workAreaSize;
  
  // 加载上次关闭时的窗口状态
  try {
    const savedState = JSON.parse(localStorage.getItem('windowState') || '{}');
    if (savedState.x !== undefined) {
      lastWindowState = savedState;
    }
  } catch (e) {
    console.warn('无法加载窗口状态', e);
  }
  
  mainWindow = new BrowserWindow({
    ...lastWindowState,
    minWidth: 1200,
    minHeight: 700,
    frame: false, // 无边框窗口
    show: false,
    backgroundColor: '#fef9f3',
    titleBarStyle: 'hidden',
    titleBarOverlay: {
      color: '#fef9f3',
      symbolColor: '#ff7043',
      height: 40
    },
    webPreferences: {
      nodeIntegration: false,
      contextIsolation: true,
      preload: path.join(__dirname, '../../preload/dist/index.cjs'),
      spellcheck: true,
      devTools: isDev
    },
    icon: path.join(__dirname, '../../resources/icons/icon.ico'),
    trafficLightPosition: { x: 16, y: 10 },
    acceptFirstMouse: true,
    autoHideMenuBar: true
  });
  
  // 加载应用
  const startUrl = isDev 
    ? 'http://localhost:5173' 
    : `file://${path.join(__dirname, '../../dist/index.html')}`;
  mainWindow.loadURL(startUrl);
  
  // 显示窗口
  mainWindow.once('ready-to-show', () => {
    mainWindow?.show();
    mainWindow?.focus();
  });
  
  // 保存窗口状态
  mainWindow.on('move', () => {
    if (!mainWindow?.isMinimized() && !mainWindow?.isMaximized()) {
      lastWindowState = mainWindow.getBounds();
    }
  });
  
  mainWindow.on('close', () => {
    if (!isQuitting) {
      lastWindowState = mainWindow?.getBounds() || lastWindowState;
      localStorage.setItem('windowState', JSON.stringify(lastWindowState));
      mainWindow?.hide();
      app.dock?.hide();
    }
  });
  
  // 最大化切换
  mainWindow.on('maximize', () => { isMaximized = true; });
  mainWindow.on('unmaximize', () => { isMaximized = false; });
  
  return mainWindow;
}

export const windowManager = {
  mainWindow: () => mainWindow,
  show: () => mainWindow?.show(),
  hide: () => mainWindow?.hide(),
  minimize: () => mainWindow?.minimize(),
  maximize: () => {
    if (isMaximized) {
      mainWindow?.unmaximize();
    } else {
      mainWindow?.maximize();
    }
  },
  close: () => mainWindow?.close(),
  isMaximized: () => isMaximized
};
```

#### 3.2.5 预加载脚本 (`electron/preload/preload.ts`)

```typescript
import { contextBridge, ipcRenderer } from 'electron';
import { IPC_CHANNELS } from '../main/ipc/channels';

// 类型定义
type WindowControl = {
  minimize: () => Promise<void>;
  maximize: () => Promise<boolean>;
  close: () => Promise<void>;
  isMaximized: () => Promise<boolean>;
};

type FileSystem = {
  selectVRM: () => Promise<string | null>;
  selectAudio: () => Promise<string | null>;
  saveDialog: (options: Electron.SaveDialogOptions) => Promise<string | null>;
  readLocal: (path: string) => Promise<string>;
};

type System = {
  notify: (title: string, body: string) => Promise<void>;
  getInfo: () => Promise<Electron.SystemInfo>;
  openPath: (path: string) => Promise<void>;
};

type LocalAI = {
  start: (config: AIConfig) => Promise<void>;
  stop: () => Promise<void>;
  getStatus: () => Promise<AIStatus>;
};

type LocalTTS = {
  speak: (text: string, voice: string) => Promise<Buffer>;
  stop: () => Promise<void>;
  getVoices: () => Promise<TTSVoice[]>;
};

type SpeechRecognition = {
  start: () => Promise<void>;
  stop: () => Promise<string>;
  isRecognizing: () => Promise<boolean>;
};

type AutoUpdater = {
  check: () => Promise<void>;
  download: () => Promise<void>;
  getStatus: () => Promise<UpdateStatus>;
};

type DataStore = {
  save: (key: string, data: unknown) => Promise<void>;
  load: <T>(key: string) => Promise<T | null>;
  delete: (key: string) => Promise<void>;
};

// 安全API暴露
contextBridge.exposeInMainWorld('electronAPI', {
  // 窗口控制
  window: {
    control: {
      minimize: () => ipcRenderer.invoke(IPC_CHANNELS.WINDOW_MINIMIZE),
      maximize: () => ipcRenderer.invoke(IPC_CHANNELS.WINDOW_MAXIMIZE),
      close: () => ipcRenderer.invoke(IPC_CHANNELS.WINDOW_CLOSE),
      isMaximized: () => ipcRenderer.invoke(IPC_CHANNELS.WINDOW_IS_MAXIMIZED)
    } as WindowControl,
    
    // 监听窗口状态变化
    onMaximizedChange: (callback: (maximized: boolean) => void) => {
      ipcRenderer.on('window:maximized-changed', (_, maximized) => callback(maximized));
      return () => ipcRenderer.removeAllListeners('window:maximized-changed');
    }
  },
  
  // 文件系统
  fs: {
    selectVRM: () => ipcRenderer.invoke(IPC_CHANNELS.FILE_SELECT_VRM),
    selectAudio: () => ipcRenderer.invoke(IPC_CHANNELS.FILE_SELECT_AUDIO),
    saveDialog: (options: Electron.SaveDialogOptions) => 
      ipcRenderer.invoke(IPC_CHANNELS.FILE_SAVE_DIALOG, options),
    readLocal: (path: string) => ipcRenderer.invoke(IPC_CHANNELS.FILE_READ_LOCAL, path)
  } as FileSystem,
  
  // 系统功能
  system: {
    notify: (title: string, body: string) => 
      ipcRenderer.invoke(IPC_CHANNELS.SYSTEM_NOTIFICATION, title, body),
    getInfo: () => ipcRenderer.invoke(IPC_CHANNELS.SYSTEM_GET_INFO),
    openPath: (path: string) => ipcRenderer.invoke(IPC_CHANNELS.SYSTEM_OPEN_PATH)
  } as System,
  
  // 本地AI服务
  localAI: {
    start: (config: AIConfig) => ipcRenderer.invoke(IPC_CHANNELS.AI_START_LOCAL, config),
    stop: () => ipcRenderer.invoke(IPC_CHANNELS.AI_STOP_LOCAL),
    getStatus: () => ipcRenderer.invoke(IPC_CHANNELS.AI_GET_STATUS)
  } as LocalAI,
  
  // 本地TTS服务
  localTTS: {
    speak: (text: string, voice: string) => 
      ipcRenderer.invoke(IPC_CHANNELS.TTS_START_LOCAL, text, voice),
    stop: () => ipcRenderer.invoke(IPC_CHANNELS.TTS_STOP_LOCAL),
    getVoices: () => ipcRenderer.invoke(IPC_CHANNELS.TTS_GET_VOICES)
  } as LocalTTS,
  
  // 语音识别
  speech: {
    start: () => ipcRenderer.invoke(IPC_CHANNELS.SPEECH_START_RECORDING),
    stop: () => ipcRenderer.invoke(IPC_CHANNELS.SPEECH_STOP_RECORDING),
    isRecognizing: () => ipcRenderer.invoke(IPC_CHANNELS.SPEECH_IS_RECOGNIZING),
    
    // 监听识别结果
    onResult: (callback: (text: string, isFinal: boolean) => void) => {
      ipcRenderer.on('speech:result', (_, text, isFinal) => callback(text, isFinal));
      return () => ipcRenderer.removeAllListeners('speech:result');
    },
    
    // 监听状态变化
    onStateChange: (callback: (state: string) => void) => {
      ipcRenderer.on('speech:state-changed', (_, state) => callback(state));
      return () => ipcRenderer.removeAllListeners('speech:state-changed');
    }
  } as SpeechRecognition,
  
  // 自动更新
  updater: {
    check: () => ipcRenderer.invoke(IPC_CHANNELS.UPDATER_CHECK),
    download: () => ipcRenderer.invoke(IPC_CHANNELS.UPDATER_DOWNLOAD),
    getStatus: () => ipcRenderer.invoke(IPC_CHANNELS.UPDATER_GET_STATUS),
    
    onStatusChange: (callback: (status: UpdateStatus) => void) => {
      ipcRenderer.on('updater:status-changed', (_, status) => callback(status));
      return () => ipcRenderer.removeAllListeners('updater:status-changed');
    }
  } as AutoUpdater,
  
  // 数据持久化
  data: {
    save: (key: string, data: unknown) => 
      ipcRenderer.invoke(IPC_CHANNELS.DATA_SAVE, key, data),
    load: <T>(key: string) => ipcRenderer.invoke(IPC_CHANNELS.DATA_LOAD, key) as Promise<T | null>,
    delete: (key: string) => ipcRenderer.invoke(IPC_CHANNELS.DATA_DELETE, key)
  } as DataStore
});
```

```typescript
  const [inputMode, setInputMode] = useState<InputMode>('text');

  const { 
    isRecording, 
    transcript, 
    startRecording, 
    stopRecording, 
    isSupported: speechSupported

  const scrollToBottom = useCallback((behavior: ScrollBehavior = 'smooth') => {
    messagesEndRef.current?.scrollIntoView({ behavior });
  }, []);
  
  useEffect(() => {
    scrollToBottom();
  }, [messages, streamingText, scrollToBottom]);
  
  // TTS回调设置
  useEffect(() => {
    ttsQueueManager.setCallbacks({
      onAudioStart: () => setIsSpeaking(true),
      onAudioEnd: () => {
        setIsSpeaking(false);
      },
      onVolumeUpdate: (volume) => {
        // 更新音量用于口型同步
      }
    });
  }, []);

      setInputValue(transcript);

    if (!inputValue.trim() || isLoading) return;
    
    const userMessage = inputValue.trim();
    setInputValue('');
    setIsLoading(true);
    setStreamingText('');

    sentenceSplitterRef.current.reset();
    
    // 添加用户消息
    const userMsg: Message = {
      id: Date.now().toString(),
      role: 'user',
      content: userMessage,
      timestamp: Date.now()
    };
    setMessages(prev => [...prev, userMsg]);
    
    try {
      // 调用AI流式服务
      await streamingAIService.streamResponse(
        userMessage,
        // onChunk: 流式文本块
        (chunk) => {
          setStreamingText(prev => prev + chunk);
          
          // 分句处理
          const sentences = sentenceSplitterRef.current.feed(chunk);
          for (const sentence of sentences) {
            if (sentence.text.trim()) {
              ttsQueueManager.addSentence(sentence.text);
            }
          }
        },
        // onComplete: 流式完成
        (fullText) => {
          // 处理剩余文本
          const remaining = sentenceSplitterRef.current.flush();

            id: Date.now().toString(),
            role: 'assistant',
            content: fullText,
            timestamp: Date.now(),
            emotion: emotionResult.emotion,
            isStreaming: false
          };
          setMessages(prev => [...prev, aiMsg]);
          setStreamingText('');
          setIsLoading(false);

        id: Date.now().toString(),
        role: 'assistant',
        content: '抱歉，我现在有些不舒服，请稍后再试...',
        timestamp: Date.now(),
        emotion: 'sad'
      };
      setMessages(prev => [...prev, errorMsg]);
    }
  }, [inputValue, isLoading]);
  
  // 键盘事件
  const handleKeyDown = (e: React.KeyboardEvent) => {

      handleSend();
    }
  };
  
  // 语音输入切换

      setInputMode('text');

      setInputMode('voice');
    }
  }, [isRecording, startRecording, stopRecording]);
  
  // 停止播放
  const handleStop = useCallback(() => {
    ttsQueueManager.stop();
    setIsSpeaking(false);
  }, []);
  
  // 快捷回复
  const handleQuickReply = (text: string) => {
    setInputValue(text);
    inputRef.current?.focus();
  };
  
  return (

              initial={{ opacity: 0, y: 20 }}
              animate={{ opacity: 1, y: 0 }}
              transition={{ duration: 0.5 }}

                <span>🌸</span>

              <h2 className={styles.welcomeTitle}>你好，我是小暖</h2>

                很高兴认识你！我会用心倾听，用温和、耐心和热情陪伴你。
```

```typescript
interface MessageBubbleProps {

    timestamp: number;
    emotion?: EmotionType;
    isStreaming?: boolean;
  };

}

const MessageBubble: React.FC<MessageBubbleProps> = ({ message, isLatest }) => {
  const [isCopied, setIsCopied] = useState(false);
  
  const isUser = message.role === 'user';
  
  // 动画配置
  const bubbleAnimation = {
    initial: { opacity: 0, y: 20, scale: 0.95 },
    animate: { opacity: 1, y: 0, scale: 1 },
    exit: { opacity: 0, y: -10, scale: 0.95 },
    transition: { type: 'spring', stiffness: 400, damping: 25 }
  };
  
  // 时间格式化
  const formatTime = (timestamp: number) => {
    const date = new Date(timestamp);
    const now = new Date();
    const diff = now.getTime() - date.getTime();
    
    if (diff < 60000) {
      return '刚刚';
    } else if (diff < 3600000) {
      return `${Math.floor(diff / 60000)}分钟前`;
    } else {
      return date.toLocaleTimeString('zh-CN', { 
        hour: '2-digit', 
        minute: '2-digit' 
      });
    }
  };
  
  // 复制文本
  const handleCopy = async () => {
    await navigator.clipboard.writeText(message.content);
    setIsCopied(true);
    setTimeout(() => setIsCopied(false), 2000);
  };
  
  // 情感图标
  const getEmotionEmoji = (emotion?: EmotionType) => {
    switch (emotion) {
      case 'happy': return '😊';
      case 'sad': return '😔';
      case 'angry': return '😠';
      case 'surprised': return '😮';
      case 'fearful': return '😨';
      case 'disgusted': return '🤢';

  };

    <motion.div
      className={`${styles.bubbleContainer} ${isUser ? styles.userBubble : styles.assistantBubble}`}

      animate={bubbleAnimation.animate}
      exit={bubbleAnimation.exit}

          <div className={styles.userAvatar}>👤</div>
        ) : (
          <div className={styles.assistantAvatar}>
            <span>🌸</span>
            {message.emotion && (
              <span className={styles.emotionBadge}>
                {getEmotionEmoji(message.emotion)}
              </span>
            )}
          </div>
        )}
      </div>
      
      {/* 气泡内容 */}
      <div className={styles.bubbleWrapper}>
        <div 
          className={`${styles.bubble} ${isUser ? styles.userBubble : styles.assistantBubble}`}
        >
          {/* 流式指示器 */}

            <div className={styles.streamingIndicator}>
              <span className={styles.streamingDot}></span>
              <span className={styles.streamingDot}></span>
              <span className={styles.streamingDot}></span>
            </div>
          )}
          
          {/* 消息内容 */}
          <div className={styles.content}>
            {message.content}
            {message.isStreaming && (
              <span className={styles.cursor}>|</span>
            )}
          </div>
          
          {/* 操作按钮 */}
          {!isUser && (
```

#### 5.2.1 方案一：Web Speech API（推荐用于Web端）

**优势：**
- 无需额外依赖
- 实时识别
- 支持中文

**限制：**
- 依赖网络
- 兼容性有限
- 隐私数据上传云端

```typescript
// hooks/useSpeechRecognition.ts
import { useState, useCallback, useEffect, useRef } from 'react';

interface SpeechRecognitionEvent extends Event {
  results: SpeechRecognitionResultList;
  resultIndex: number;
}

interface SpeechRecognitionErrorEvent extends Event {
  error: string;
  message: string;
}

declare global {
  interface Window {
    SpeechRecognition: typeof SpeechRecognition;
    webkitSpeechRecognition: typeof SpeechRecognition;
  }
}

interface UseSpeechRecognitionOptions {
  language?: string;
  continuous?: boolean;
  interimResults?: boolean;
  maxAlternatives?: number;
}

interface UseSpeechRecognitionReturn {
  isRecording: boolean;
  isSupported: boolean;
  transcript: string;
  interimTranscript: string;
  confidence: number;
  startRecording: () => void;
  stopRecording: () => void;
  resetTranscript: () => void;
  error: string | null;
}

export function useSpeechRecognition(
  options: UseSpeechRecognitionOptions = {}
): UseSpeechRecognitionReturn {
  const {
    language = 'zh-CN',
    continuous = true,
    interimResults = true,
    maxAlternatives = 1
  } = options;
  
  const [isRecording, setIsRecording] = useState(false);
  const [transcript, setTranscript] = useState('');
  const [interimTranscript, setInterimTranscript] = useState('');
  const [confidence, setConfidence] = useState(0);
  const [error, setError] = useState<string | null>(null);
  
  const recognitionRef = useRef<SpeechRecognition | null>(null);
  
  // 检查支持性
  const isSupported = typeof window !== 'undefined' && 
    ('SpeechRecognition' in window || 'webkitSpeechRecognition' in window);
  
  // 初始化
  useEffect(() => {
    if (!isSupported) return;
    
    const SpeechRecognition = window.SpeechRecognition || window.webkitSpeechRecognition;
    recognitionRef.current = new SpeechRecognition();
    
    recognitionRef.current.continuous = continuous;
    recognitionRef.current.interimResults = interimResults;
    recognitionRef.current.maxAlternatives = maxAlternatives;
    recognitionRef.current.lang = language;
    
    // 结果处理
    recognitionRef.current.onresult = (event: SpeechRecognitionEvent) => {
      let finalTranscript = '';
      let interimText = '';
      
      for (let i = event.resultIndex; i < event.results.length; i++) {
        const result = event.results[i];
        
        if (result.isFinal) {
          finalTranscript += result[0].transcript;
          setConfidence(result[0].confidence);
        } else {
          interimText += result[0].transcript;
        }
      }
      
      setTranscript(prev => prev + finalTranscript);
      setInterimTranscript(interimText);
    };
    
    // 错误处理
    recognitionRef.current.onerror = (event: SpeechRecognitionErrorEvent) => {
      setError(event.error);
      setIsRecording(false);
      
      // 自动重启（除致命错误外）
      if (event.error !== 'no-speech' && event.error !== 'aborted') {
        setTimeout(() => {
          try {
            recognitionRef.current?.start();
          } catch (e) {
            console.error('重启识别失败:', e);
          }
        }, 1000);
      }
    };
    
    // 结束处理
    recognitionRef.current.onend = () => {
      setIsRecording(false);
      
      // 如果是主动停止则不重启
      if (!recognitionRef.current) return;
      
      // 连续模式下自动重启
      if (continuous && error !== 'aborted') {
        try {
          recognitionRef.current.start();
          setIsRecording(true);
        } catch (e) {
          console.error('重启识别失败:', e);
        }
      }
    };
    
    return () => {
      if (recognitionRef.current) {
        recognitionRef.current.abort();
        recognitionRef.current = null;
      }
    };
  }, [isSupported, language, continuous, interimResults, maxAlternatives, error]);
  
  // 开始录音
  const startRecording = useCallback(() => {
    if (!isSupported || !recognitionRef.current || isRecording) return;
    
    setError(null);
    setInterimTranscript('');
    
    try {
      recognitionRef.current.start();
      setIsRecording(true);
    } catch (e) {
      console.error('启动语音识别失败:', e);
      setError('启动失败');
    }
  }, [isSupported, isRecording]);
  
  // 停止录音
  const stopRecording = useCallback(() => {
    if (!recognitionRef.current || !isRecording) return;
    
    try {
      recognitionRef.current.stop();
      setIsRecording(false);
    } catch (e) {
      console.error('停止语音识别失败:', e);
    }
  }, [isRecording]);
  
  // 重置
  const resetTranscript = useCallback(() => {
    setTranscript('');
    setInterimTranscript('');
    setConfidence(0);
  }, []);
  
  return {
    isRecording,
    isSupported,
    transcript,
    interimTranscript,
    confidence,
    startRecording,
    stopRecording,
    resetTranscript,
    error
  };
}
```

### 5.3 语音处理流程

```typescript
// services/speechProcessing.ts
class SpeechProcessor {
  // 音频预处理
  async preprocessAudio(audioBlob: Blob): Promise<Float32Array> {
    // 1. 解码音频
    const audioContext = new AudioContext();
    const arrayBuffer = await audioBlob.arrayBuffer();
    const audioBuffer = await audioContext.decodeAudioData(arrayBuffer);
    
    // 2. 重采样到16kHz（Whisper要求）
    const targetSampleRate = 16000;
    const resampled = this.resample(audioBuffer, targetSampleRate);
    
    // 3. 归一化
    const normalized = this.normalize(resampled);
    
    // 4. 降噪（可选）
    const denoised = await this.removeNoise(normalized);
    
    return denoised;
  }
  
  // 重采样
  private resample(audioBuffer: AudioBuffer, targetSampleRate: number): Float32Array {
    const sourceData = audioBuffer.getChannelData(0);
    const sourceSampleRate = audioBuffer.sampleRate;
    const ratio = sourceSampleRate / targetSampleRate;
    const targetLength = Math.ceil(sourceData.length / ratio);
    const result = new Float32Array(targetLength);
    
    for (let i = 0; i < targetLength; i++) {
      const srcIndex = i * ratio;
      const srcIndexFloor = Math.floor(srcIndex);
      const srcIndexCeil = Math.min(srcIndexFloor + 1, sourceData.length - 1);
      const t = srcIndex - srcIndexFloor;
      
      // 线性插值
      result[i] = sourceData[srcIndexFloor] * (1 - t) + 
                  sourceData[srcIndexCeil] * t;
    }
    
    return result;
  }
  
  // 归一化
  private normalize(data: Float32Array): Float32Array {
    let max = 0;
    for (let i = 0; i < data.length; i++) {
      const abs = Math.abs(data[i]);
      if (abs > max) max = abs;
    }
    
    if (max === 0) return data;
    
    const result = new Float32Array(data.length);
    for (let i = 0; i < data.length; i++) {
      result[i] = data[i] / max;
    }
    
    return result;
  }
  
  // 标点恢复
  async restorePunctuation(text: string): Promise<string> {
    // 使用简单的规则或调用NLP服务
    return text
      .replace(/([。！？…])\s*/g, '$1\n')
      .replace(/([，；：])\s*/g, '$1 ')
      .replace(/\n{2,}/g, '\n')
      .trim();
  }
  
  // 文本规范化
  normalizeText(text: string): string {
    return text
      .replace(/\s+/g, ' ')  // 合并空白字符
      .replace(/[^\u4e00-\u9fa5a-zA-Z0-9\s，。！？、]/g, '')  // 移除非标准字符
      .trim();
  }
  
  // 敏感词过滤
  filterSensitiveWords(text: string): string {
    const sensitiveWords = ['暴力', '色情', '赌博']; // 敏感词库
    let result = text;
    
    for (const word of sensitiveWords) {
      result = result.replace(new RegExp(word, 'g'), '*'.repeat(word.length));
    }
    
    return result;
  }
}
```

```typescript
const useOptimizedRenderLoop = (callback: () => void) => {
  const frameRef = useRef<number>(0);
  const lastTimeRef = useRef<number>(0);
  const fps = 60;
  const frameInterval = 1000 / fps;
  
  const animate = useCallback((time: number) => {
    frameRef.current = requestAnimationFrame(animate);
    
    const deltaTime = time - lastTimeRef.current;
    
    if (deltaTime >= frameInterval) {
      lastTimeRef.current = time - (deltaTime % frameInterval);
      callback();
    }
  }, [callback, deltaTime, frameInterval, frameRef, lastTimeRef]);
  
  useEffect(() => {
    frameRef.current = requestAnimationFrame(animate);
    return () => {
      if (frameRef.current) {
        cancelAnimationFrame(frameRef.current);
      }
    };
  }, [animate]);
};
```

```typescript
// 1. 请求缓存
const createCachedFetch = () => {
  const cache = new Map<string, { data: unknown; timestamp: number }>();
  const CACHE_DURATION = 5 * 60 * 1000; // 5分钟
  
  return async <T>(url: string, options?: RequestInit): Promise<T> => {
    const cacheKey = `${url}${JSON.stringify(options)}`;
    const cached = cache.get(cacheKey);
    
    if (cached && Date.now() - cached.timestamp < CACHE_DURATION) {
      return cached.data as T;
    }
    
    const response = await fetch(url, options);
    const data = await response.json();
    
    cache.set(cacheKey, { data, timestamp: Date.now() });
    return data as T;
  };
};
```
