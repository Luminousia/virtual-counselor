# VRM动画文件说明

## 支持的文件格式

### .vrma 文件
VRM Animation (.vrma) 文件是专门为VRM模型设计的动画文件格式。
如果你有 `.vrma` 格式的待机动画文件（如 `idle_loop.vrma`），可以将其放在 `public` 文件夹中。

## 程序化动画

如果没有 `.vrma` 文件，系统会自动使用程序化的待机动画，包括：
- **呼吸动画**：胸部轻微上下移动
- **轻微摆动**：身体和头部的自然小动作
- **自然姿态**：让模型看起来更生动

## 获取VRM动画文件

你可以从以下来源获取或创建VRM动画：

1. **VRM Animation Hub**: 查找社区分享的动画文件
2. **VRoid Studio**: 创建自己的动画
3. **Blender**: 使用VRM插件导出动画

## 使用方法

如果将来实现.vrma文件加载功能，只需将动画文件放在 `public` 文件夹即可自动加载。
